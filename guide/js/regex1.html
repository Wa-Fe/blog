# 匹配某个范围内的所有的值

var reg = /((?<=\[img])(.+?)(?=\[\/img]))/g

这个表达式匹配的是 [img] xxxx [/img] 之间的内容，因为中括号的特殊性，需要在结尾的地方括号后main添加一个转义自负 \，如果不包含特殊的字符，则此表达式为 /((?<=\A)(.+?)(?=\C))/g, 匹配 AxxxxC 之间的内容
